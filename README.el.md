> ⚠️ **ΠΕΙΡΑΜΑΤΙΚΗ ΥΛΟΠΟΙΗΣΗ** - Πρόκειται για μια πειραματική, εκπαιδευτική επανυλοποίηση του Apple Macintosh System 7. ΔΕΝ είναι ολοκληρωμένο προϊόν και δεν πρέπει να θεωρείται λογισμικό έτοιμο για παραγωγική χρήση.

Μια επανυλοποίηση ανοιχτού κώδικα του Apple Macintosh System 7 για σύγχρονο υλικό x86, με δυνατότητα εκκίνησης μέσω GRUB2/Multiboot2. Αυτό το έργο στοχεύει στην αναδημιουργία της κλασικής εμπειρίας Mac OS, τεκμηριώνοντας παράλληλα την αρχιτεκτονική του System 7 μέσω ανάλυσης αντίστροφης μηχανικής.

## 🎯 Κατάσταση Έργου

**Τρέχουσα κατάσταση**: Ενεργή ανάπτυξη με ~94% βασική λειτουργικότητα ολοκληρωμένη

### Τελευταίες Ενημερώσεις (Νοέμβριος 2025)

#### Βελτιώσεις Sound Manager ✅ ΟΛΟΚΛΗΡΩΜΕΝΟ
- **Βελτιστοποιημένη μετατροπή MIDI**: Κοινόχρηστος βοηθός `SndMidiNoteToFreq()` με πίνακα αναζήτησης 37 εγγραφών (C3-B5) και εναλλακτικό μηχανισμό βασισμένο σε οκτάβες για πλήρες εύρος MIDI (0-127)
- **Υποστήριξη ασύγχρονης αναπαραγωγής**: Πλήρης υποδομή callbacks τόσο για αναπαραγωγή αρχείων (`FilePlayCompletionUPP`) όσο και για εκτέλεση εντολών (`SndCallBackProcPtr`)
- **Δρομολόγηση ήχου βασισμένη σε κανάλια**: Σύστημα προτεραιοτήτων πολλαπλών επιπέδων με ελέγχους σίγασης και ενεργοποίησης
  - Κανάλια προτεραιότητας 4 επιπέδων (0-3) για δρομολόγηση εξόδου υλικού
  - Ανεξάρτητοι έλεγχοι σίγασης και ενεργοποίησης ανά κανάλι
  - Η `SndGetActiveChannel()` επιστρέφει το ενεργό κανάλι υψηλότερης προτεραιότητας
  - Σωστή αρχικοποίηση καναλιού με ενεργοποιημένη σημαία εξ ορισμού
- **Υλοποίηση ποιότητας παραγωγής**: Όλος ο κώδικας μεταγλωττίζεται χωρίς σφάλματα, δεν εντοπίστηκαν παραβιάσεις malloc/free
- **Commits**: 07542c5 (βελτιστοποίηση MIDI), 1854fe6 (ασύγχρονα callbacks), a3433c6 (δρομολόγηση καναλιών)

#### Επιτεύγματα Προηγούμενων Συνεδριών
- ✅ **Φάση Προχωρημένων Χαρακτηριστικών**: Βρόχος επεξεργασίας εντολών Sound Manager, σειριοποίηση στυλ πολλαπλών εκτελέσεων, εκτεταμένες δυνατότητες MIDI/σύνθεσης
- ✅ **Σύστημα Αλλαγής Μεγέθους Παραθύρων**: Διαδραστική αλλαγή μεγέθους με σωστή διαχείριση πλαισίου, grow box και καθαρισμό επιφάνειας εργασίας
- ✅ **Μετάφραση Πληκτρολογίου PS/2**: Πλήρης αντιστοίχιση scancode set 1 σε κωδικούς πλήκτρων Toolbox
- ✅ **HAL Πολλαπλών Πλατφορμών**: Υποστήριξη x86, ARM και PowerPC με καθαρή αφαίρεση

## 📊 Πληρότητα Έργου

**Συνολική Βασική Λειτουργικότητα**: ~94% ολοκληρωμένη (εκτίμηση)

### Πλήρως Λειτουργικά ✅

- **Επίπεδο Αφαίρεσης Υλικού (HAL)**: Πλήρης αφαίρεση πλατφόρμας για x86/ARM/PowerPC
- **Σύστημα Εκκίνησης**: Επιτυχής εκκίνηση μέσω GRUB2/Multiboot2 σε x86
- **Σειριακή Καταγραφή**: Καταγραφή βασισμένη σε ενότητες με φιλτράρισμα κατά τον χρόνο εκτέλεσης (Error/Warn/Info/Debug/Trace)
- **Βάση Γραφικών**: Framebuffer VESA (800x600x32) με πρωτόγονα QuickDraw συμπεριλαμβανομένης λειτουργίας XOR
- **Απόδοση Επιφάνειας Εργασίας**: Γραμμή μενού System 7 με πολύχρωμο λογότυπο Apple, εικονίδια και μοτίβα επιφάνειας εργασίας
- **Τυπογραφία**: Γραμματοσειρά bitmap Chicago με απόδοση pixel-perfect και σωστό kerning, εκτεταμένο Mac Roman (0x80-0xFF) για ευρωπαϊκούς τονισμένους χαρακτήρες
- **Διεθνοποίηση (i18n)**: Τοπικοποίηση βασισμένη σε πόρους με 34 γλώσσες (Αγγλικά, Γαλλικά, Γερμανικά, Ισπανικά, Ιταλικά, Πορτογαλικά, Ολλανδικά, Δανικά, Νορβηγικά, Σουηδικά, Φινλανδικά, Ισλανδικά, Ελληνικά, Τουρκικά, Πολωνικά, Τσεχικά, Σλοβακικά, Σλοβενικά, Κροατικά, Ουγγρικά, Ρουμανικά, Βουλγαρικά, Αλβανικά, Εσθονικά, Λετονικά, Λιθουανικά, Μακεδονικά, Μαυροβουνιακά, Ρωσικά, Ουκρανικά, Ιαπωνικά, Κινεζικά, Κορεατικά, Χίντι), Locale Manager με επιλογή γλώσσας κατά την εκκίνηση, υποδομή κωδικοποίησης πολλαπλών bytes για CJK
- **Font Manager**: Υποστήριξη πολλαπλών μεγεθών (9-24pt), σύνθεση στυλ, ανάλυση FOND/NFNT, αποθήκευση LRU
- **Σύστημα Εισόδου**: Πληκτρολόγιο και ποντίκι PS/2 με πλήρη προώθηση συμβάντων
- **Event Manager**: Συνεργατική πολυδιεργασία μέσω WaitNextEvent με ενοποιημένη ουρά συμβάντων
- **Memory Manager**: Εκχώρηση βασισμένη σε ζώνες με ενσωμάτωση διερμηνέα 68K
- **Menu Manager**: Πλήρη αναπτυσσόμενα μενού με παρακολούθηση ποντικιού και SaveBits/RestoreBits
- **Σύστημα Αρχείων**: HFS με υλοποίηση B-tree, παράθυρα φακέλων με απαρίθμηση VFS
- **Window Manager**: Μεταφορά, αλλαγή μεγέθους (με grow box), διαστρωμάτωση, ενεργοποίηση
- **Time Manager**: Ακριβής βαθμονόμηση TSC, ακρίβεια μικροδευτερολέπτου, έλεγχος γενεάς
- **Resource Manager**: Δυαδική αναζήτηση O(log n), κρυφή μνήμη LRU, ολοκληρωμένη επικύρωση
- **Gestalt Manager**: Πληροφορίες συστήματος πολλαπλών αρχιτεκτονικών με ανίχνευση αρχιτεκτονικής
- **TextEdit Manager**: Πλήρης επεξεργασία κειμένου με ενσωμάτωση πρόχειρου
- **Scrap Manager**: Κλασικό πρόχειρο Mac OS με υποστήριξη πολλαπλών τύπων
- **Εφαρμογή SimpleText**: Πλήρης επεξεργαστής κειμένου MDI με αποκοπή/αντιγραφή/επικόλληση
- **List Manager**: Στοιχεία ελέγχου λίστας συμβατά με System 7.1 με πλοήγηση πληκτρολογίου
- **Control Manager**: Τυπικά στοιχεία ελέγχου και γραμμές κύλισης με υλοποίηση CDEF
- **Dialog Manager**: Πλοήγηση πληκτρολογίου, δακτύλιοι εστίασης, συντομεύσεις πληκτρολογίου
- **Segment Loader**: Φορητό σύστημα φόρτωσης τμημάτων 68K ανεξάρτητο ISA με μετατόπιση
- **Διερμηνέας M68K**: Πλήρης αποστολή εντολών με 84 χειριστές opcodes, όλοι οι 14 τρόποι διευθυνσιοδότησης, πλαίσιο εξαιρέσεων/trap
- **Sound Manager**: Επεξεργασία εντολών, μετατροπή MIDI, διαχείριση καναλιών, callbacks
- **Device Manager**: Διαχείριση DCE, εγκατάσταση/αφαίρεση οδηγών και λειτουργίες I/O
- **Οθόνη Εκκίνησης**: Πλήρες UI εκκίνησης με παρακολούθηση προόδου, διαχείριση φάσεων και οθόνη εκκίνησης
- **Color Manager**: Διαχείριση κατάστασης χρωμάτων με ενσωμάτωση QuickDraw

### Μερικώς Υλοποιημένα ⚠️

- **Ενσωμάτωση Εφαρμογών**: Ο διερμηνέας M68K και ο segment loader είναι ολοκληρωμένοι· απαιτείται δοκιμή ενσωμάτωσης για επαλήθευση ότι πραγματικές εφαρμογές εκτελούνται
- **Διαδικασίες Ορισμού Παραθύρων (WDEF)**: Βασική δομή εγκατεστημένη, μερική αποστολή
- **Speech Manager**: Πλαίσιο API και διέλευση ήχου μόνο· ο μηχανισμός σύνθεσης ομιλίας δεν έχει υλοποιηθεί
- **Χειρισμός Εξαιρέσεων (RTE)**: Η επιστροφή από εξαίρεση είναι μερικώς υλοποιημένη (επί του παρόντος σταματά αντί να αποκαθιστά το περιβάλλον)

### Δεν Έχει Υλοποιηθεί Ακόμη ❌

- **Εκτύπωση**: Κανένα σύστημα εκτύπωσης
- **Δικτύωση**: Καμία λειτουργικότητα AppleTalk ή δικτύου
- **Βοηθητικά Εφαρμογίδια Γραφείου (Desk Accessories)**: Μόνο πλαίσιο
- **Προχωρημένος Ήχος**: Αναπαραγωγή δειγμάτων, μίξη (περιορισμός PC speaker)

### Υποσυστήματα Χωρίς Μεταγλώττιση 🔧

Τα ακόλουθα διαθέτουν πηγαίο κώδικα αλλά δεν είναι ενσωματωμένα στον πυρήνα:
- **AppleEventManager** (8 αρχεία): Ανταλλαγή μηνυμάτων μεταξύ εφαρμογών· σκόπιμα αποκλεισμένο λόγω εξαρτήσεων pthread ασύμβατων με αυτόνομο περιβάλλον
- **FontResources** (μόνο header): Ορισμοί τύπων πόρων γραμματοσειρών· η πραγματική υποστήριξη γραμματοσειρών παρέχεται από το μεταγλωττισμένο FontResourceLoader.c

## 🏗️ Αρχιτεκτονική

### Τεχνικές Προδιαγραφές

- **Αρχιτεκτονική**: Πολλαπλές αρχιτεκτονικές μέσω HAL (x86, ARM, PowerPC έτοιμο)
- **Πρωτόκολλο Εκκίνησης**: Multiboot2 (x86), bootloaders ειδικοί ανά πλατφόρμα
- **Γραφικά**: VESA framebuffer, 800x600 @ 32-bit χρώμα
- **Διάταξη Μνήμης**: Ο πυρήνας φορτώνεται στη φυσική διεύθυνση 1MB (x86)
- **Χρονισμός**: Ανεξάρτητος αρχιτεκτονικής με ακρίβεια μικροδευτερολέπτου (RDTSC/καταχωρητές χρονομέτρου)
- **Επιδόσεις**: Αστοχία κρυφής μνήμης πόρων <15μs, ευστοχία κρυφής μνήμης <2μs, απόκλιση χρονομέτρου <100ppm

### Στατιστικά Κώδικα

- **225+ αρχεία πηγαίου κώδικα** με ~57.500+ γραμμές κώδικα
- **145+ αρχεία κεφαλίδων** σε 28+ υποσυστήματα
- **69 τύποι πόρων** εξαγόμενοι από το System 7.1
- **Χρόνος μεταγλώττισης**: 3-5 δευτερόλεπτα σε σύγχρονο υλικό
- **Μέγεθος πυρήνα**: ~4,16 MB
- **Μέγεθος ISO**: ~12,5 MB

## 🔨 Μεταγλώττιση

### Απαιτήσεις

- **GCC** με υποστήριξη 32-bit (`gcc-multilib` σε 64-bit)
- **GNU Make**
- **Εργαλεία GRUB**: `grub-mkrescue` (από `grub2-common` ή `grub-pc-bin`)
- **QEMU** για δοκιμές (`qemu-system-i386`)
- **Python 3** για επεξεργασία πόρων
- **xxd** για μετατροπή δυαδικών
- *(Προαιρετικό)* **powerpc-linux-gnu** εργαλειοθήκη cross-compilation για builds PowerPC

### Εγκατάσταση σε Ubuntu/Debian

```bash
sudo apt-get install build-essential gcc-multilib grub-pc-bin xorriso qemu-system-x86 python3 vim-common
```

### Εντολές Μεταγλώττισης

```bash
# Μεταγλώττιση πυρήνα (x86 εξ ορισμού)
make

# Μεταγλώττιση για συγκεκριμένη πλατφόρμα
make PLATFORM=x86
make PLATFORM=arm        # απαιτεί ARM bare-metal GCC
make PLATFORM=ppc        # πειραματικό· απαιτεί εργαλειοθήκη PowerPC ELF

# Δημιουργία εκκινήσιμου ISO
make iso

# Μεταγλώττιση με όλες τις γλώσσες
make LOCALE_FR=1 LOCALE_DE=1 LOCALE_ES=1 LOCALE_JA=1 LOCALE_ZH=1 LOCALE_KO=1 LOCALE_RU=1 LOCALE_UK=1 LOCALE_PL=1 LOCALE_CS=1 LOCALE_SQ=1 LOCALE_BG=1 LOCALE_HR=1 LOCALE_DA=1 LOCALE_NL=1 LOCALE_ET=1 LOCALE_FI=1 LOCALE_EL=1 LOCALE_HU=1 LOCALE_IS=1 LOCALE_IT=1 LOCALE_LV=1 LOCALE_LT=1 LOCALE_MK=1 LOCALE_ME=1 LOCALE_NO=1 LOCALE_PT=1 LOCALE_RO=1 LOCALE_SK=1 LOCALE_SL=1 LOCALE_SV=1 LOCALE_TR=1 LOCALE_HI=1

# Μεταγλώττιση με μία επιπλέον γλώσσα
make LOCALE_FR=1

# Μεταγλώττιση και εκτέλεση σε QEMU
make run

# Καθαρισμός αρχείων μεταγλώττισης
make clean

# Εμφάνιση στατιστικών μεταγλώττισης
make info
```

## 🚀 Εκτέλεση

### Γρήγορη Εκκίνηση (QEMU)

```bash
# Τυπική εκτέλεση με σειριακή καταγραφή
make run

# Χειροκίνητα με επιλογές
qemu-system-i386 -cdrom system71.iso -serial file:/tmp/serial.log -display sdl -vga std -m 256M
```

### Επιλογές QEMU

```bash
# Με σειριακή έξοδο κονσόλας
qemu-system-i386 -cdrom system71.iso -serial stdio -display sdl -m 256M

# Χωρίς γραφικά (headless)
qemu-system-i386 -cdrom system71.iso -serial stdio -display none -m 256M

# Με αποσφαλμάτωση GDB
make debug
# Σε άλλο τερματικό: gdb kernel.elf -ex "target remote :1234"
```

4. Υλοποίηση καθαρού δωματίου (χωρίς πρωτότυπο πηγαίο κώδικα της Apple)

**Το πρωτότυπο ROM και λογισμικό του System 7 παραμένει ιδιοκτησία της Apple Inc.**

## 🙏 Ευχαριστίες

- **Apple Computer, Inc.** για τη δημιουργία του αρχικού System 7
- **Συγγραφείς Inside Macintosh** για την εκτενή τεκμηρίωση
- **Κοινότητα διατήρησης κλασικών Mac** για τη διατήρηση της πλατφόρμας ζωντανής
- **68k.news και Macintosh Garden** για αρχεία πόρων

## 📊 Στατιστικά Ανάπτυξης

- **Γραμμές Κώδικα**: ~57.500+ (συμπεριλαμβανομένων 2.500+ για τον segment loader)
- **Χρόνος Μεταγλώττισης**: ~3-5 δευτερόλεπτα
- **Μέγεθος Πυρήνα**: ~4,16 MB (kernel.elf)
- **Μέγεθος ISO**: ~12,5 MB (system71.iso)
- **Μείωση Σφαλμάτων**: 94% της βασικής λειτουργικότητας λειτουργεί
- **Κύρια Υποσυστήματα**: 28+ (Font, Window, Menu, Control, Dialog, TextEdit, κ.λπ.)

## 🔮 Μελλοντική Κατεύθυνση

**Προγραμματισμένες Εργασίες**:

- Ολοκλήρωση βρόχου εκτέλεσης διερμηνέα M68K
- Προσθήκη υποστήριξης γραμματοσειρών TrueType
- Πόροι γραμματοσειρών bitmap CJK για απόδοση Ιαπωνικών, Κινεζικών και Κορεατικών
- Υλοποίηση πρόσθετων στοιχείων ελέγχου (πεδία κειμένου, αναδυόμενα μενού, ρυθμιστές)
- Εγγραφή σε δίσκο για σύστημα αρχείων HFS
- Προχωρημένες δυνατότητες Sound Manager (μίξη, δειγματοληψία)
- Βασικά βοηθητικά εφαρμογίδια γραφείου (Αριθμομηχανή, Σημειωματάριο)
